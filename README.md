# Autism Health Companion App

**Version:** 0.1.0-alpha  
**Status:** Alpha Testing

## Overview
An Android app designed specifically for people with autism to track health metrics, manage sensory sensitivities, mood tracking, and integrate with Samsung Health.

## Features
- **Health Tracking**: Steps, heart rate, sleep monitoring
- **Mood & Sensory Tracking**: Log emotional states and sensory preferences
- **Samsung Health Integration**: Seamless data sync
- **Autism-Friendly UI**: 
  - High contrast options
  - Reduced animations
  - Simple, clear navigation
  - Text scaling support
  - Dark mode support
- **Routine Management**: Daily routines and reminders
- **Emergency Contacts**: Quick access during overwhelm

## Installation
1. Download the APK from the Releases section
2. Enable installation from unknown sources on your device
3. Install the APK
4. Grant requested permissions

## Requirements
- Android 8.0 (API 26) or higher
- Samsung Health app installed (for health tracking features)
- 50MB free storage

This is an early alpha release. Please report bugs and suggest features via GitHub Issues.
